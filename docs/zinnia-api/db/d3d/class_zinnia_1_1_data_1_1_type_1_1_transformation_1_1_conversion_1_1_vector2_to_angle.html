Detailed Description

Transforms a Vector2 value to a float angle.

The origin angle (i.e. zero) is set to the top by default to mimic a clock with 12 o'clock being the 0 angle.

Vector2[0f, 0f] -> unit.degrees -> origin.Vector2(0f, 1f) = 0f Vector2[1f, 0f] -> unit.degrees -> origin.Vector2(0f, 1f) = 90f Vector2[0f, -1f] -> unit.degrees -> origin.Vector2(0f, 1f) = 180f
