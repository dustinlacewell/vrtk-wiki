Detailed Description

Transforms a Vector3 to a Vector2 and allows mapping of the relevant coordinates.

Vector3(1f, 2f, 3f) -> XToXAndYToYExcludeZ -> Vector2(1f, 2f) Vector3(1f, 2f, 3f) -> XToYAndYToXExcludeZ -> Vector2(2f, 1f) Vector3(1f, 2f, 3f) -> XToYAndZToXExcludeY -> Vector2(3f, 1f)
