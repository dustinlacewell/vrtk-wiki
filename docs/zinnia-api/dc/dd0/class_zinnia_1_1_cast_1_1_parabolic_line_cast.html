◆ CollisionCheckFrequency

int Zinnia.Cast.ParabolicLineCast.CollisionCheckFrequency
getset

The number of points along the parabolic line to check for an early cast collision. Useful if the parabolic line is appearing to clip through locations. 0 won't make any checks and it will be capped at SegmentCount.

The higher the number, the more CPU intensive the checks become.
