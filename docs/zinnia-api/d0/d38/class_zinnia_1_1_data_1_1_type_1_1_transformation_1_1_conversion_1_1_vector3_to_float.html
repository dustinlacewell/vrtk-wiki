Detailed Description

Transforms a Vector3 to a float and allows mapping of the relevant coordinates.

Vector3(2f, 3f, 6f) -> ExtractX -> 2f Vector3(2f, 3f, 6f) -> ExtractY -> 3f Vector3(2f, 3f, 6f) -> ExtractZ -> 6f Vector3(2f, 3f, 6f) -> ExtractMagnitude -> 7f Vector3(2f, 3f, 6f) -> ExtractSqrMagnitude -> 49f
