Detailed Description

Transforms a float by remapping from a range to a new range.

2f -> From(0f, 10f), To(0f, 1f), Mode(Lerp) = 0.2f 2f -> From(0f, 10f), To(1f, 0f), Mode(Lerp) = 0.8f 2f -> From(10f, 0f), To(0f, 1f), Mode(Lerp) = 0.8f 2f -> From(10f, 0f), To(1f, 0f), Mode(Lerp) = 0.2f 2f -> From(0f, 10f), To(0f, 1f), Mode(SmoothStep) = 0.104f
