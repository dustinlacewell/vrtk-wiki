Detailed Description

Transforms a Vector2 to a Vector3 and allows mapping of the relevant coordinates.

Vector2(1f, 2f) -> XToXAndYToY -> Vector3(1f, 2f, 0f) Vector2(1f, 2f) -> XToXAndYToZ -> Vector3(1f, 0f, 2f) Vector2(1f, 2f) -> XToYAndYToX -> Vector3(2f, 1f, 0f)
