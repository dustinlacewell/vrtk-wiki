enum Zinnia.Pointer.PointerElement.Visibility

The visibility of a PointerElement.

Enumerator
OnWhenPointerActivated 

The PointerElement will only be visible when the ObjectPointer is activated.

AlwaysOn 

The PointerElement will always be visible regardless of the ObjectPointer state.

AlwaysOff 

The PointerElement will never be visible regardless of the ObjectPointer state.
